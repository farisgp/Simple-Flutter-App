# Simple Flutter App

Simple Flutter App is a basic mobile application built using **Flutter**.  
This project was created as a learning exercise to understand the fundamentals of Flutter development, including user interface design, navigation, and state management.

The application demonstrates how to build a simple mobile interface using Flutter's widget-based architecture.

---

## Features

- Simple and responsive user interface
- Flutter widget-based layout
- Basic navigation between screens
- Clean and organized project structure
- Beginner-friendly Flutter implementation

---

## Tech Stack

The application is built using the following technologies:

- **Flutter**
- **Dart**
- **Material Design**

---
## Installation

### 1. Clone the repository
```bash
git clone https://github.com/farisgp/Simple-Flutter-App.git
```

### 2. Navigate to the project folder
```bash
cd Simple-Flutter-App
```

### 3. Install dependencies
```bash
flutter pub get
```

### 4. Run the application
```bash
flutter run
```

## Requirements

Make sure the following tools are installed on your system:
- Flutter SDK
- Dart
- Android Studio or Visual Studio Code
- Android Emulator or Physical Device

## To verify your Flutter installation:
```bash
flutter doctor
```
